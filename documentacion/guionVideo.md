# Guion
Autogestión es una herramienta de gestión de concesionario.

Tiene modos claro y oscuro.

## catalogo
Este es el catálogo, donde se muestran todos los vehículos disponibles para la venta.

Tiene botones para ver más información sobre un vehículo.

## masInformacion
En esta ventana se muestra la información detallada de un vehículo.

Al pusar el botón comprar se abre el formulario de compra.

Se piden los datos del cliente y se realiza la compra.
